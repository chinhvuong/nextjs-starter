# SVN-Gilroy Font Setup

This directory contains the SVN-Gilroy font files used in the project.

## Required Font Files

To use SVN-Gilroy fonts, download and place the following files in this directory:

- `SVN-Gilroy-Regular.woff2` - Regular weight (400)
- `SVN-Gilroy-Medium.woff2` - Medium weight (500)
- `SVN-Gilroy-XBold.woff2` - Extra Bold weight (800)

## Download Sources

You can download SVN-Gilroy fonts from:

1. **OnlineWebFonts**: https://www.onlinewebfonts.com/fonts/SVN_Gilroy_Regular
2. **Fonnts**: https://fonnts.com/svn-gilroy/
3. **Exfont**: https://exfont.com/font-family/svn-gilroy

## Font Usage

The fonts are configured in `src/shared/fonts/svn-gilroy.ts` and can be used with:

- `.font-svn-gilroy-regular` - Regular weight
- `.font-svn-gilroy-medium` - Medium weight
- `.font-svn-gilroy-xbold` - Extra Bold weight

## Note

If font files are not present, the system will fall back to system fonts (Inter, system fonts).
